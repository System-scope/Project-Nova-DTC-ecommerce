## User Account & Post-Purchase Experience

This section outlines the user account functionalities and post-purchase features that enable users to manage their profile, orders, and preferences across the platform.

---

### 1. User Profile Management

Users can:

- View and edit personal details
- Manage contact information

---

### 2. Address Management

Users can:

- Add multiple delivery addresses
- Edit or delete saved addresses
- Mark an address as default

**System Behavior:**
- Default address is auto-selected during checkout
- Users can change the selected address before proceeding
- If no address exists → user is required to add a new address

**UX Consideration:**
Pre-selecting a default address reduces friction and speeds up checkout.

---

### 3. Saved Payment Methods

Users can:

- Save preferred payment methods
- Use saved options for quicker transactions

---

### 4. Wishlist Management

Users can:

- Add/remove items to wishlist
- Move items from wishlist to cart

**Purpose:**
- Helps users track desired products
- Supports future purchase intent and retention

---

### 5. Order History

Users can:

- View past orders
- Access order details including:
  - Items purchased
  - Pricing breakdown
  - Payment method
  - Invoice

---

### 6. Order Tracking

Users can track order status:

- Placed
- Confirmed
- Packed
- Shipped
- Out for Delivery
- Delivered

**Access Points:**
- Order confirmation page
- Order history section

---

### 7. Returns & Refunds (User Side)

Users can initiate and track returns from the order history.

**Return Initiation:**
- Select return reason (mandatory)
- Upload product image (mandatory)
- Add additional comments (optional)

**System Behavior:**
- Return request cannot be submitted without required inputs
- Request is created with "Return Requested" status

---

**Refund Tracking:**

Users can track refund status as:

- Return Requested  
- Return Approved  
- Refund Processing  
- Refunded  

**Refund Behavior:**
- Refund is processed after approval
- Amount is credited to original payment method

---

### Key Focus Areas

- Seamless post-purchase experience
- Faster checkout through saved data
- Transparent order and refund tracking
- User convenience and retention
